# Mercado Liebre

Proyecto Integrador, abarca los temas:

- Levantar un servidor web con Express.  
- Estructura de un sitio web.  
- Agregar estilos.  
- Diseño Adaptativo.  
- Posicionamiento avanzado con Flexbox.  
- Formularios.  
- Animando elementos.  
- Diseño MVC.  
- Motor de templates.  
- CRUD.  
- Middlewares.  
- Session y Cookies.  
- Base de datos MySQL.  
- Sequelize.
